    On Mac OS, we strongly recommend moving SigmoID.app to the Applications folder before the first launch. We also recommend to launch SigmoID for the first time only after all required software is installed.
    SigmoID v.2 works on OS X (10.10 and up) and 64-bit Linux distributions (Ubuntu 16 and Mint 16 and up were tested). Other Linux distributions may work provided Xojo requirements are met.
    Full list of platform-specific system requirements is located at Xojo web site.
    SigmoID relies heavily on Python and will have severely limited functionality without it. The various python scripts included with SigmoID require python 3 with a matching version of Biopython.

    Requirements listed below should be installed in the following order:

1) OpenMPI or MPICH to allow MEME and MeShClust to use multiple threads. Absence of OpenMPI will be especially noticeable in the de novo TFBS inference pipeline: meme will run in a single thread slowing the pipeline down significantly. MeShClust will not run at all, so one of the clustering steps will be skipped which will decrease the quality of the motifs inferred.
2) Install MEME Suite. Depending on your system set-up, MEME may require additional libraries and perl modules (html output generation may not work without those).
3) Install MeShClust.
4) Install NCBI Entrez Direct (EDirect).
5) HMMER.

    The paths to all external programs can be changed in the preferences. SigmoID will try to find the paths to the versions installed on your system (via the PATH variable). Any missing programs will be reported in the main window at launch.

Optional software
   
    ChIPMunk is not required for standard operation of SigmoID, but may be very helpful for inference of nonpalindromic motifs. To use ChIPmunk, please set the path to the chipmunk.jar file in the preferences.

Promoter inference requires Clustal Omega and BioProspector. The paths to the executables of these programs should also be set in the preferences.
